이름: 대여 내역 목록 테이블
형태: 데이터 테이블 (Data Table)
구분: React 컴포넌트
디자인 프레임워크: Tailwind CSS
기능 여부: 정렬 / 페이징 없음 (기본 렌더링 테이블)

[상단 구성]
- 상단 타이틀: "대여 내역 총 603건"
  - "603건"은 실제 데이터의 `.length`로 계산되며 파란색 텍스트

[테이블 헤더]
다음 순서로 구성된 12개 열:
1. NO (번호, 정렬 가능)
2. 자산 ID
3. 요청인
4. 도착 희망일
5. 도착지
6. 콘텐츠
7. 대여기간
8. 자산 이름
9. 자산 유형
10. 재고 수량 (예: 14 EA)
11. 요청 수량
12. 재고 위치

[디자인 및 Tailwind 클래스 요구]
